PROJECT TITLE: TechSupport
TechSupport - the DodgySoft Technical support system.

This project is part of the material for the book

  
Objects First with Java - 
A Practical Introduction using BlueJ
   Sixth edition
   David J. Barnes and Michael Kölling
   
Pearson Education, 2016

This project is discussed in chapter 6.



PURPOSE OF PROJECT: 
This project aims to implements a technical support system for customers of 
the DodgySoft software company. 
Users can describe their software problems and 
get advice instantly!

 The idea is based on Eliza - 
a famous program described by Joseph Weizenbaum 
in 1966.
VERSION or DATE: 04.06.2018
HOW TO START THIS PROJECT: To start this program, create a SupportSystem object and execute the "start" 
method.



AUTHORS:  Mayya Bareeva, Huong Phan, Omar Gabarine
USER INSTRUCTIONS: Start describing your problem by typing in the terminal window.

 
Be clear and precise while typing in. If the programme does not know the answer, restate your question
on other words.
